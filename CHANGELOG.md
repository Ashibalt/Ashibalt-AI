# Changelog

Все заметные изменения в расширении **Ashibalt AI** задокументированы здесь.

## [0.2.1] - 2026-02-22

### Changed / Изменено

- **Настройки** переделаны в нижний drawer-панель (slide-up снизу, полная ширина, `border-radius: 14px 14px 0 0`) — больше не модальное окно; убрана сильная визуальная разделённость навигации и контента
- **Область ввода** полностью переработана по дизайну «Version A»: единый panel с бордером вокруг всего блока, компактная строка метрик (20px, разделители-трубы, моно-шрифт), плашка изменений (26px, кнопки Сохранить/Откатить) стала первым вложенным элементом панели, `textarea` облегчён (font-weight 300, font-size 12.5px), нижняя bar 28px с border-top
- Кнопки режима/модели — минималистичный стиль без pill-границы (hover только фон), соответствие Version A `.sel`
- Кнопка отправки уменьшена до 22×22px (`border-radius: 4px`, border вместо glow)
- Выпадающее меню режима — компактные строки (`4px 8px` padding, `border-radius: 3px`, размер 11.5px)
- Удалён шрифт Geist из стека (недоступен в CSP webview); используется `var(--vscode-font-family)` с системным fallback
- `pending-changes-dashboard` перемещён внутрь `input-container` — единый панельный вид

## [0.2.0] - 2026-07-16

### Added / Добавлено

- **Автовыбор провайдера** — при отправке запроса расширение автоматически определяет и выбирает наиболее подходящего провайдера для выбранной модели. Если модель доступна у нескольких провайдеров, приоритет отдаётся локальным (Ollama), затем по скорости ответа. Ручной выбор по-прежнему доступен.
- **Параллельный вызов инструментов (parallel tool calls)** — включён для всех моделей в режиме агента. Модель может вызывать несколько инструментов одновременно в рамках одной итерации, что значительно ускоряет выполнение сложных задач (чтение файлов, запуск команд, поиск и т.д.).

### Changed / Изменено

- Единая цветовая схема Variant C применена ко всему интерфейсу: обновлены CSS-переменные, цвета поверхностей, акценты, шрифт
- Настройки переделаны из выдвижной панели в центрированное модальное окно с дизайном сплит-панели.
- Блок ввода (чат + метрики + дашборд изменений) — убраны видимые рамки, лёгкий минимальный дизайн без визуальной тяжести

# Ashibalt AI

> **Բետա** — ակտիվ մշակման փուլում է։ Արձագանքները ողջունվում են։

---

## Ինչ է սա

**Ashibalt AI** -ը Visual Studio Code-ի լիարժեք AI ծրագրավորման գործակալ է։ Այն ոչ միայն պատասխանում է հարցերին, այլ կարող է ինքնուրույն խմբագրել ֆայլերը, գործարկել տերմինալի հրամաններ, որոնել ձեր նախագծում, ախտորոշել սխալները և լուծել բարդ խնդիրներ։

## Հնարավորություններ

- **Գործակալի ռեժիմ** — ինքնուրույն կոդի խմբագրում, ֆայլերի ստեղծում, տերմինալի հրամաններ, խնդիրների լուծում
- **Զրուցի ռեժիմ** — միայն կարդալու AI օգնական ձեր կոդի համատեքստով
- **4 մատակարար** — Ollama (տեղային), OpenRouter, Mistral, DeepSeek
- **Մոդելների դիտարկիչ** — որոնում և ավելացում UI-ից
- **Սնեպշոտների համակարգ** — յուրաքանչյուր խմբագրում ստեղծում է վերականգնվող սնեպշոտ
- **11 գործիք** — `read_file`, `edit_file`, `create_file`, `delete_file`, `list_files`, `search`, `terminal`, `write_to_terminal`, `read_terminal_output`, `diagnose`, `fetch_url`
- **Շարահյուսության ստուգում** — tree-sitter վերլուծություն 14+ լեզուների համար
- **Համատեքստի կառավարում** — ավտոմատ սեղմում, մինչև 128K
- **Չափումներ** — տոկենների իրական օգտագործում, քեշի ցուցադրում
- **Նիստեր** — զրույցի պատմության պահպանում և որոնում

## Արագ մեկնարկ

1. Տեղադրեք ընդլայնումը VS Code Marketplace-ից
2. Բացեք Ashibalt կողմնային վահանակը
3. Բացեք ⚙️ Կարգավորումներ, ընտրեք մատակարար և մուտքագրեք API բանալին
4. Ընտրեք մոդելը և սկսեք ծրագրավորել

## Գաղտնիություն

- API բանալիները պահվում են տեղային
- Տվյալները փոխանցվում են միայն ձեր և AI մատակարարի միջև
